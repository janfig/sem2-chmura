## start.sh
Służy do uruchomienia projektu lokalnie

## Użyte komendy gh cli:
### Utworzenie repozytoriów:
`gh repo create`
### Dodanie sekretów:
`gh secret list`
### Uruchomienie workflowa:
`gh workflow run`
### Sprawdzenie statusu workflowa:
`gh workflow view`
### Szczegóy poszczególnego uruchomienia workflowa:
`gh run view`
